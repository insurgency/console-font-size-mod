## Building & Installing Manually

```shell
$ sudo apt install -y git make python3-pip
$ pip3 install --upgrade --user vpk
$ git clone https://github.com/insurgency/console-font-size-mod.git
$ cd ./console-font-size-mod/
$ make install
```
